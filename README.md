# GithubFollowers

## About
- 100% programmtic UI
- MVC
- RESTful API
- Support for light and dark mode
- No third-party libraries
- Custom UIAlerts
- Child ViewControllers
- Empty States

## Technologies Used
- User Defaults
- CollectionView
- NSCache
- Diffable Data Source
- Async/Await For Network Calls
- Search Functionality 
- Result
